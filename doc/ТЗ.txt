Задача: Разработка системы навигации для автономного робота
Описание задачи
Целью данного задания является разработка основы системы навигации для автономного робота, который должен
самостоятельно передвигаться в заданной среде. Ваша задача — создать алгоритм, который позволит роботу определять
кратчайший и безопасный путь между двумя точками в пространстве с препятствиями.

Задания
1. Моделирование пространства:
Сгенерировать карту пространства в виде сетки, где каждая ячейка может быть свободной или занятой (препятствие).
Размер карты и расположение препятствий задаются случайным образом при инициализации.
2. Поиск пути:
Реализуйте алгоритм A* (A-star) для нахождения кратчайшего пути от стартовой точки до цели. Учтите, что робот не может
проходить через препятствия.
3. Оптимизация маршрута:
После нахождения пути, оптимизируйте его таким образом, чтобы минимизировать количество поворотов, необходимых роботу
для его прохождения.

4. Визуализация:
Визуализируйте карту пространства, путь робота и препятствия. Используйте библиотеку Matplotlib или любую другую
библиотеку Python для визуализации.

Требования
- Используйте Python версии 3.6 или выше;
- Для реализации алгоритма A* и работы с данными можно использовать стандартные библиотеки Python;
- Код должен быть читаем, хорошо структурирован и сопровождаться комментариями;
- Предусмотрите обработку ошибок, например, отсутствие пути из-за препятствий.

Критерии оценки
- Корректность и эффективность реализованных алгоритмов;
- Качество и читаемость кода;
- Оригинальность подхода к оптимизации маршрута;
- Визуализация и наглядность представления результатов.
